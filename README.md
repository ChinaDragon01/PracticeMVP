# Android MVP模式--简单实用示例（BMI） 
<br></br>
## MVP简介
> MVP全称：Model、View、Presenter； 
> 
- View：负责视图部分展示
- Model：负责数据的请求、解析、过滤等数据层操作。
- Presenter：View和Model交互的桥梁。
